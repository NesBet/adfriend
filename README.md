## ADFRIEND  

# About  
Adfriend is a simple browser extension that replaces ad blocks  
in websites with a variety of statements.  
These statements include:  
    - Bible verses  
    - Fun facts  
    - Activities
    - otivational quotes  

Feel free to use it and give a star.
